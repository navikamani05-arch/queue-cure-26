# Queue Cure '26

## Problem Statement
Many clinics still use paper tokens and manual queue management. Patients have no visibility into waiting time, and receptionists manage queues manually.

## Solution
Queue Cure is a real-time patient queue management system that helps receptionists manage queues and allows patients to track their waiting status instantly.

## Features
- Add Patient
- Automatic Token Generation
- Call Next Token
- Real-Time Updates
- Estimated Waiting Time
- Doctor Availability Status
- Voice Announcements
- Queue Progress Tracking
- Kid Zone

## Tech Stack
- React.js
- Firebase Realtime Database
- JavaScript
- HTML
- CSS

## Installation

1. Install dependencies
npm install

2. Start application
npm start

3. Open
http://localhost:3000
